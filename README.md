# Library Book Management System

This project implements a Library Book Management System using Python.

## Features
- Add books with ID, title, and author
- Borrow and return books
- Generate a library status report

## Technologies Used
- Python
- unittest
- Git

## Agile Process
The project was developed using three sprints:
- Sprint-1: Book Registration
- Sprint-2: Borrow and Return
- Sprint-3: Library Report
